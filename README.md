# aitool
#### 介绍 
  界面精美的ai开发框架，可以快速接入chatgpt、deepseek、pika、sora等模型，开发出一套可以商业运营的AI工具  
#### 软件架构 
  后端：springboot 若依开发框架 前端：uniapp vue
